# 📦 Inventory Management System

A full-stack **Inventory Management System** built to simulate how businesses manage products, users, stock, and suppliers efficiently.

This project demonstrates my ability to design and develop secure, scalable backend systems and connect them to a functional frontend.

---

## 🚀 Tech Stack

### Backend
- Java (Spring Boot)
- RESTful APIs
- JWT Authentication & Authorization

### Frontend (This is front end)
- HTML
- CSS
- JavaScript

### Database
- Microsoft SQL Server

---

## 🔐 Features

- ✅ Secure authentication using JWT  
- ✅ Role-based access control (Admin & Staff)  
- ✅ User management (Create, Update, Delete users)  
- ✅ Product management  
- ✅ Category management  
- ✅ Supplier management  
- ✅ Stock tracking  
- ✅ Protected routes with authorization checks  
- ✅ Clean and structured UI navigation  

---

## 🧠 What I Learned

This project helped me gain practical experience in:

- Building secure REST APIs using Spring Boot  
- Implementing authentication and authorization with JWT  
- Designing relational databases in SQL Server  
- Connecting frontend UI with backend APIs  
- Handling user roles and permissions  
- Writing maintainable and scalable code
